						REDE e Segurança

	Foi implementado uma rede LAN com internet da UniFi Network de alta qualidade com protocolo TCP/IP com uma camada de Firewall.
	A rede será distribuida para todas as salas, na utilização de um switch distribuindo os dados que serão enviado pela topologia estrela.
                                               
                                             Meio físico
	Foi usado cabos com qualidade Cat 5e de pares trançados e blindagem metálica, para melhoria de transferência de dados e reduzindo interferências.

                                            Segurança de dados
        Os dados são armazenados em um sistema NAS com configuração RAID 5 para que não haja perda de dados redundantes para a empresa.

					     Equipamentos
	Kit Ubiquiti para redes de pequenas empresas 

















Foi estabelecida uma solução LAN com internet da UniFi Network de alta qualidade para os conjuntos de salas,
recebendo no roteador e mandando para o switch que distribui para todas as areas da empresa usando a topologia
estrela. Usamos também um AP WI-FI para que todos possam usar a internet sem precisar ser por meios fisícos.
	Foi implementado um NAS com configuração de RAID 5 na rede para poder ter todo os dados armazenados com segurança,
sendo assim todos os dados redundantes para a empresa são salvos via rede.
